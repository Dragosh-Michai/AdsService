# AvitoMiniApp - Сервис частных объявлений

## Описание проекта
WPF-приложение для работы с частными объявлениями (купля-продажа, аренда, услуги). 
Разработано в рамках учебной практики по дисциплине "Информационные системы и программирование".

## Функциональность
- ✅ Просмотр объявлений с фильтрацией и поиском
- ✅ Авторизация и регистрация пользователей
- ✅ Управление собственными объявлениями
- ✅ Отслеживание завершённых сделок с подсчётом прибыли
- ✅ Резервное копирование базы данных

## Технологии
- .NET Framework 4.8
- WPF (C#)
- Entity Framework
- SQL Server
- MVVM Pattern
